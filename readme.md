# About application. share-life

## Description

This application is social network were you can posting your interests or your life and get feedback.

## What you can do?

-   Registration/Login
-   Create/Remove/Search posts
-   Create/Remove comments
-   Update information for user
-   View posts in home and user page
-   Follow users

## What i used?

### Client

-   React
-   Redux
-   React context
-   React-router-dom
-   Typescript
-   Sass (modules)
-   Socket.io-client

### Server

-   Node.js
-   Express
-   Multer
-   Socket.io
-   PostgreSQL
-   JWT
-   Bcrypt
-   SQL
